解决 The user operation is waiting for "Building workplace" to complete

1.问题描述：
今天在写单元测试的时候遇到了一个问题，点击 Run as C> Junit Test 后，
eclipse 弹出一个框，一直在跑着什么东西，上面写着：
The user operation is waiting for “Building workplace” to complete，
等了好长时间也没见进度条前进了多少。 

2.解决办法：
点击菜单栏上的 Project 选项，把 Build Automactically 前面的勾去掉就好了。
